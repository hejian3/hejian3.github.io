java 内存泄漏

Java

Java 内存溢出的三种方式

  1. PermGen space 方法区溢出 加载过多的class
  2. Java heap space 堆溢出 堆中对象过多
  3. unable to create new native thread 系统内存太少导致无法创建本地线程

如何解决

  1. PermGen space 1G内存环境下java jvm 的参数设置参考：JAVA_OPTS="-server -Xms800m -Xmx800m -XX:PermSize=64M -XX:MaxNewSize=256m - XX:MaxPermSize=128m -Djava.awt.headless=true"

  2. Java heap space

    • 先确定是否有内存泄漏（内存检测MAT）
    • 然后修改JVM参数(参考方法去泄漏)
  3. unable to create new native thread 不要将JVM参数设置过大而导致本地内存不足，或者可以调整栈内存
